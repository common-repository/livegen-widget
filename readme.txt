LiveGen widget by Global-Roam.

== Description ==

We are pleased to be able to work with the RenewEconomy team to help people understand how much renewable 
generation is currently contributing to electricity supplies in various locations around the world.

It shows, for each location around the world, how much electricity is being 
generated by each fuel type, using data published by various sources.

If you don't already have an API key you can get one [here](http://widgetadmin.generationdata.info/register/livegen). 
Alternately, you can move your mouse over the widget and click the **EMBED** button.

== Installation ==

If you have already received an email with a link to the embedding instructions, please follow them to find your API key.

1. Install the plugin as usual
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[livegen-widget token="your-api-token"]` on your page

If you don't already have an API key you can get one [here](http://widgetadmin.generationdata.info/register/livegen). 
Alternately, you can move your mouse over the widget and click the **EMBED** button.
